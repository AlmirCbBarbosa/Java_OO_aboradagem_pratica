Java tipos primitivos:
    boolean -> representa valores lógico: true, false

    tipos de inteiros:
        byte -> ocupa 1 byte na memória valores: -128 a 127
        short -> 
        int ->
        long -> ocupa 8 bytes na memoria: -9223372036854775808 e 9223372036854775807

    tipos de ponto flutuante:
        float -> precisão simples
        double -> precisão dupla

    char -> é utilizado para armazenar um caracteres, que deve ficar entre aspas simples

    String -> representa uma cadeia de caracteres, mas não é do tipo primitiva, mas sim um classe.

operadores
    aritméticos:
        + -> soma;
        - -> subtração;
        * -> multiplicação;
        / -> divisão;
        % -> retorna o resto de uma divisão.

    relacionais:
        < -> menor que;
        > -> maior que;
        <= -> menor igual que;
        >= -> maior igual que;
        == -> igual a;
        = -> diferente que.

    operadores lógicos:
        && -> and
        || -> or

    = -> é operador de atribuição;
    equals -> é operador de comparação do tipo/classe String.
